Installation de l'application de gestion Quincaillerie :

1. Créez une base de données MySQL nommée `quincaillerie`.
2. Importez le fichier `quincaillerie.sql` dans PhpMyAdmin.
3. Placez tous les fichiers dans `htdocs/quincaillerie-app` (XAMPP) ou `www/` (Laragon).
4. Lancez `http://localhost/quincaillerie-app/` dans votre navigateur.
5. Connectez-vous via l'interface de connexion avec les identifiants préremplis.
